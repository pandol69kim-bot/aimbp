## 주요 기능

### ✅ 커버 생성
- 앨범 내에서 직접 커버 생성 가능
- 3가지 비율 동시 생성 (1:1, 16:9, 9:16)
- 자동 상태 업데이트 (pending → completed)

### ✅ 커버 표시

### ✅ 다운로드
- 각 비율별 이미지 개별 다운로드
- 링크 방식 (외부 URL)

## 사용 방법

### 앨범에서 커버 생성하기

1. **앨범 상세 페이지 접속**
   - http://localhost:3001/albums/{album-id}

2. **커버 생성 버튼 클릭**
   - 우상단 "커버 생성" 버튼

3. **정보 입력**
   - 장르 선택 (Pop, K-Pop, Rock 등)
   - 분위기 선택 (몽환적, 에너제틱 등)
   - 키워드 입력 (쉼표로 구분)

4. **커버 생성**
   - "커버 생성" 버튼 클릭
   - 약 5초 대기

5. **커버 확인**
   - 3가지 비율 이미지 자동 표시
   - 각 이미지 호버 시 다운로드 가능
