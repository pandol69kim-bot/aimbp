## 작업 내용

**업로드된 MD 파일의 원본 형식을 그대로 유지하면서 표시하도록 변경했습니다.**

사용자가 작성한 마크다운 형식(## Verse, ## Chorus 등)이 그대로 보여지므로, 원본 가사를 그대로 확인할 수 있습니다.

### Before
```
┌─────────────────────────────────┐

│                                 │

```

```
┌─────────────────────────────────┐
│ # Nature's Fury: Echoes...      │

│                                 │
│ ## Chorus                       │
│ Cry the storm...                │

│                                 │

마크다운 형식 그대로 표시
```

## ✨ 개선사항

### 1️⃣ 마크다운 형식 유지
```
# 제목
## 섹션명
가사 내용
```

### 2️⃣ 메타데이터 표시
```
# Nature's Fury: Echoes of Despair
Genre: Rock • Mood: 어둡고 강렬한
```

### 3️⃣ 코드 에디터처럼 표시
```
- 흑색 배경 (코드 에디터 느낌)
- 고정폭 폰트 (font-mono)
- 색상 구분
```

### 4️⃣ 원본 보존
```
사용자가 작성한 형식 그대로
```

## 💻 코드 변경사항

### 레이아웃 변경
```typescript
// Before
<div className="space-y-6">
  {/* 각 섹션을 분리된 박스로 표시 */}
</div>

// After
<div className="bg-gray-950/50 rounded-lg p-6 font-mono text-xs">
  {/* 코드 에디터처럼 마크다운 형식 그대로 표시 */}
</div>
```

### 제목 및 메타데이터
```typescript
// 새 추가
<p className="text-base font-bold text-primary-300 mb-2">
  # {lyrics.title}
</p>
<p className="text-gray-400">
  {lyrics.prompt_genre && `Genre: ${lyrics.prompt_genre}`}
  {lyrics.prompt_mood && `Mood: ${lyrics.prompt_mood}`}
</p>
```

### 섹션 표시
```typescript
// Before
<p className="text-xs font-semibold text-primary-400 mb-3 uppercase">VERSE</p>

// After
<p className="text-sm font-semibold text-primary-300 mb-2">## Verse</p>
// ↑ 마크다운 형식 유지
```

### 스타일 개선
```
배경: bg-gray-950/50    (코드 에디터 어두운 배경)
폰트: font-mono         (고정폭 폰트)
크기: text-xs           (가사 전체를 한 화면에)
```

## 🔄 MD 파일 예시

### 업로드한 형식
```markdown
# Nature's Fury: Echoes of Despair

**Genre:** Rock
**Mood:** 어둡고 강렬한

---

## Verse

Sky's on fire, the world's gone mad,
In a storm of tears, the earth feels sad.
...

## Chorus

Cry the storm, scream the pain,
We're lost in the hurricane.
...

## Bridge

We chase the wind, shadows of the past,
Praying for mercy that never lasts.
...

## Hook

Is there a way to heal this plight?
In nature's wrath, we fight the night.
```

### 시스템에서 표시되는 형식
```
# Nature's Fury: Echoes of Despair
Genre: Rock • Mood: 어둡고 강렬한
─────────────────────────────────

## Verse

Sky's on fire, the world's gone mad,
In a storm of tears, the earth feels sad.
...

## Chorus

Cry the storm, scream the pain,
We're lost in the hurricane.
...
```
