## 작업 내용

앨범 커버 생성 목록에 이미지 미리보기 및 다운로드 기능을 개선하여 사용자 경험을 향상시켰습니다.

### 주요 개선사항

1. **이미지 미리보기 강화**
   - 각 생성 항목에서 3가지 비율(1:1, 16:9, 9:16)의 썸네일 표시
   - 그리드 레이아웃으로 한눈에 보기 쉽게 배치
   - 각 이미지 비율 라벨 표시

2. **다운로드 기능 개선**
   - 호버 시 각 이미지에 다운로드 버튼 표시
   - 개별 비율별 이미지 다운로드 가능
   - Primary 컬러 버튼으로 시각적 강조

3. **상태별 UI 표현**
   - **완료 상태**: 이미지 그리드 + 다운로드 가능
   - **처리 중 상태**: 애니메이션 로딩 스피너 표시
   - **실패 상태**: 실패 메시지 표시

4. **목록 레이아웃 최적화**
   - 더 컴팩트한 카드 높이 (space-y-3)
   - 헤더 정보 축소로 이미지 공간 확대
   - 호버 효과 추가 (이미지 스케일 업)
   - 비율 라벨이 이미지 하단에 오버레이로 표시
