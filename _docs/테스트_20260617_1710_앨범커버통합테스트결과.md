```

요청: test@aimbp.com / test1234
```

**상태: 통과**

### 2️⃣ 앨범 생성
```
POST /api/v1/albums
요청:
  - title: "Test Album Cover"
  - description: "Test album for cover generation"

응답:
  - id: dd663194-2648-475c-89ea-e350ca8567c6
  - status: draft
  - cover_url: null (아직 설정 안 함)
```

**상태: 통과** ✅

### 3️⃣ 커버 생성 (앨범 연결)
```
POST /api/v1/cover/generate
요청:
  - prompt_genre: "rock"
  - prompt_mood: "dark"
  - prompt_keywords: "electric, night, power"
  - ai_model: "dalle-3"
  - album_id: "dd663194-2648-475c-89ea-e350ca8567c6"

응답:
  - 3개의 Cover 객체 반환
  - 각각 size: 1:1, 16:9, 9:16
  - 초기 상태: pending
```

**상태: 통과** ✅

### 4️⃣ 커버 생성 완료 (상태 변경)
```
5초 대기 후 상태 확인

결과:
┌─────────────────────────────────────────────────┐
│ Size: 1:1   | Status: completed | ✅             │
│ URL: https://picsum.photos/seed/cover11/...    │
├─────────────────────────────────────────────────┤
│ Size: 16:9  | Status: completed | ✅             │
│ URL: https://picsum.photos/seed/cover169/...   │
├─────────────────────────────────────────────────┤
│ Size: 9:16  | Status: completed | ✅             │
│ URL: https://picsum.photos/seed/cover916/...   │
└─────────────────────────────────────────────────┘
```

**상태: 통과** ✅

### 5️⃣ 앨범 상세 조회
```
GET /api/v1/albums/{album_id}
응답:
  - album_id: dd663194-2648-475c-89ea-e350ca8567c6
  - title: "Test Album Cover"
  - description: "Test album for cover generation"
  - cover_url: null (아직 수동 설정 필요)
  - tracks: [] (비어있음)
```

**상태: 통과** ✅
(cover_url은 자동으로 설정되지 않음. Frontend에서 커버 목록 별도 조회)

### 6️⃣ 앨범별 커버 목록 조회
```
GET /api/v1/cover
필터링: album_id == "dd663194-2648-475c-89ea-e350ca8567c6"

결과:
  - 3개의 커버 모두 완벽히 필터링됨
  - 각각 album_id 정확히 설정됨
  - image_url 정상 저장됨
```

**상태: 통과** ✅

## 발견된 이슈

### ✅ (해결됨) 커버 상태 업데이트
**문제:** 커버 생성 후 status가 pending으로 남아있음  
**원인:** Background task 처리 시간  
**해결:** 5초 대기 후 확인 시 completed로 변경됨  
**상태:** 정상 작동 ✅
