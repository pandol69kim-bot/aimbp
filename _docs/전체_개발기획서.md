# AIMBP (AI Music Business Platform) 전체 개발기획서

## 프로젝트 개요

### 목표
AI 기반의 음악 제작 플랫폼으로, 사용자가 AI를 활용하여 음악을 생성, 편집, 마스터링하고 배포할 수 있는 통합 솔루션 제공

### 핵심 가치 제안
- **AI 음악 생성**: Suno AI를 통한 자동 작곡
- **전문가 수준의 편집**: 가사, 보컬, 편곡, 마스터링
- **플랫폼 최적화**: Spotify, Apple Music 등 스트리밍 플랫폼 자동 최적화
- **비용 효율성**: 전문 음악가 없이도 고품질 음악 제작 가능

### 3.2 음악 생성 (Music Generation)

#### 흐름
```
사용자 입력 (제목, 장르, BPM, 분위기)
    ↓
Suno AI API 호출
    ↓
백그라운드에서 음악 생성 (진행 상황 폴링)
    ↓
완료 시 S3에 저장
    ↓
UI에서 재생 및 다운로드 가능
```

#### 데이터 모델
```python
class Track:
    id: UUID
    user_id: UUID (FK)
    title: str
    genre: str
    bpm: int (선택)
    mood: str (선택)
    lyrics_id: UUID (FK, 선택)
    file_url: str (완료 후)
    status: str (pending/processing/completed/failed)
    ai_service: str (suno/mureka/udio)
    task_id: str (AI 서비스 작업 ID)
    error_message: str (실패 시)
    created_at: datetime
```

#### API 엔드포인트
```
POST   /music/generate      - 음악 생성 시작
GET    /music              - 사용자의 음악 목록
GET    /music/{id}         - 음악 상세 조회
DELETE /music/{id}         - 음악 삭제
```

```python
from passlib.context import CryptContext

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

# 저장
hashed_pwd = pwd_context.hash(password)

# 검증
is_valid = pwd_context.verify(password, hashed_pwd)
```
