## 📋 작업 내용

AIMBP 플랫폼의 음악 앨범을 정식으로 발행한 후, 발행된 앨범은 수정 불가능하게 잠금하고 동시에 사용자가 발행된 앨범을 다운로드할 수 있는 기능에 대한 상세한 개발 계획서를 작성했습니다.

이 기능은 다음을 포함합니다:
- 앨범 발행 (DRAFT → PUBLISHED)
- 발행된 앨범 수정 금지
- 앨범 전체 ZIP 다운로드
- 개별 음원 다운로드
