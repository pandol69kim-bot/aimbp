## 주요 결정 사항

### 1. 디자인 철학: "다크 럭셔리"
프로젝트의 현재 스타일을 "다크 럭셔리" 테마로 정의했습니다. 이는:
- 고급스럽고 세련된 외관
- 매우 어두운 배경 (#0f0f0f)
- 글래스모픽 효과 (반투명 + 블러)
- 목적있는 색상 사용

### 2. 기존 시스템 활용
프로젝트에 이미 구현된 것을 모두 문서화했습니다:
- Tailwind CSS 기반
- CVA (Class Variance Authority) 패턴
- 3개 주요 색상 (Primary, Secondary, Accent)
- 반응형 설계
- 커스텀 유틸리티 클래스 (glass, glass-dark, text-gradient)

### 3. 표준 재사용 컴포넌트
- `frontend/src/components/ui/` 폴더의 컴포넌트 사용 권장
- 새로운 컴포넌트 생성 전 기존 것 확인 필수
- 컴포넌트별 변형(variant) 시스템 활용

### 4. 접근성 우선
- 모든 상호작용 요소에 포커스 상태 필수
- WCAG AA 표준 준수
- 색상 대비 최소 4.5:1
- Semantic HTML 사용

### 5. 문서 2-레벨 구조
- **DESIGN_SYSTEM.md**: 상세한 가이드 (참고용)
- **UI-QUICK-REFERENCE.md**: 빠른 체크리스트 (개발 중 참고용)
- 개발자는 상황에 따라 선택 사용

## 기술 정보

### 프로젝트의 UI 스택
- **Framework**: Next.js (React)
- **CSS**: Tailwind CSS (유틸리티 퍼스트)
- **컴포넌트 패턴**: CVA (Class Variance Authority)
- **아이콘**: Lucide React
- **다크 모드**: CSS 변수 + Tailwind

### 색상 시스템 상세
```typescript
Primary   (보라):    #7c3aed (RGB: 124 58 237)
Secondary (파란):    #3b82f6 (RGB: 59 130 246)
Accent    (시안):    #06b6d4 (RGB: 6 182 212)
Background:         #0f0f0f (매우 어두움)
Card Background:    #1a1a2e (어두운 카드)
Text (주요):        #ffffff (흰색)
Text (보조):        #9ca3af (회색)
Border:             rgba(255, 255, 255, 0.1)
```
