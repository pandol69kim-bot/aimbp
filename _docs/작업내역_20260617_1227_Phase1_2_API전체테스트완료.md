## 이전 세션에서 수정된 버그

| 버그 | 해결책 |
|------|--------|
| `pydantic[email]` 누락 → 422 오류 | requirements.txt에 `pydantic[email]`, `email-validator==2.2.0` 추가 |
| `passlib`/`bcrypt` 버전 충돌 | `bcrypt==4.0.1` 고정 |
| `next.config.ts` Next.js 14 미지원 | `next.config.js` (CommonJS)로 변환 |

| vocal 생성 FK 위반 (vocal_library 비어있음) | `init_db()`에서 `_seed_vocal_library()` 호출, 4개 샘플 시드 |
