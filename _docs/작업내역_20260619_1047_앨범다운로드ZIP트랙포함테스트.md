## 작업 내용

앨범 다운로드 ZIP 파일에 트랙이 정상 포함되는지 테스트 및 검증 완료.

## 문제 진단

- 트랙 `file_url`이 `https://musicfile.removeai.ai/...` 외부 URL로 저장됨
- 해당 URL은 `transfer-encoding: chunked` 스트리밍 방식으로, 서버 사이드 요청 시 0 bytes 반환 (Cloudflare 보호 또는 스트리밍 전용)
- 기존 완료된 4개 트랙 URL 모두 만료 상태 확인

## 변경된 파일

- `backend/app/api/albums.py`:
  - `import httpx` 추가
  - `_fetch_bytes()` 헬퍼 함수 추가 (로컬 파일 및 외부 URL 모두 지원)
  - 커버·트랙 모두 해당 함수로 통일
  - 앨범·트랙 제목 특수문자(`/`, `\`) 이스케이프 처리

- `backend/app/api/music.py`:
  - `import os, httpx` 추가
  - `_download_and_save_track()` 비동기 함수 추가
  - 콜백 수신 시 (`audio`, `first_audio`, `text`) `BackgroundTasks`로 즉시 파일 다운로드 → `/app/uploads/tracks/{id}.mp3` 로컬 저장 → DB `file_url` 업데이트
