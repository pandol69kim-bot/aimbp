## Backend 코드 구조

### 작사 (ai_lyrics.py)
```python
# OpenAI 호출
generate_lyrics_openai() → gpt-4o-mini
  ├─ has_openai 확인
  ├─ JSON 응답 파싱
  └─ verse, chorus, bridge, hook 추출

# Claude 호출
generate_lyrics_claude() → claude-haiku-4-5
  ├─ has_anthropic 확인
  ├─ Markdown 제거
  └─ JSON 파싱
```

### 작곡 (ai_music.py)
```python
# Mureka 호출
_call_mureka_generate() → POST /v1/song/generate
  ├─ task_id 반환
  └─ 백그라운드 폴링 시작

_poll_mureka_task() → GET /v1/song/{task_id}
  └─ status + audio_url 반환
```

## 사용 흐름

```
1. 프론트엔드에서 작사 생성 요청
   ↓
2. Backend OpenAI API 호출
   ↓
3. JSON 응답 파싱
   ↓
4. DB에 저장 후 반환
   ↓
5. 프론트엔드에서 작곡 생성 시 lyrics_id 포함
   ↓
6. Backend Mureka API 호출
   ↓
7. task_id로 백그라운드 폴링 시작
   ↓
8. 완료되면 file_url 업데이트
   ↓
9. 프론트엔드에서 음악 재생
```
