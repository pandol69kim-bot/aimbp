## 주요 결정 사항

### gpt-image-1 vs DALL-E 3 차이점

| 항목 | DALL-E 3 | GPT Image 1 |
|---|---|---|
| 모델 ID | `dall-e-3` | `gpt-image-1` |
| 16:9 사이즈 | `1792x1024` | `1536x1024` |
| 9:16 사이즈 | `1024x1792` | `1024x1536` |
| quality 옵션 | `standard` / `hd` | `low` / `medium` / `high` |
| 응답 형식 | URL | base64 (`b64_json`) |

### 응답 처리
- gpt-image-1은 기본적으로 `b64_json` 형식으로 반환
- `data:image/png;base64,{b64}` 형식으로 변환하여 저장
- `url` 형식으로 반환된 경우 폴백 처리

### 폴백 순서
```
gpt-image-1 선택 시:
  gpt-image-1 API → 실패 시 Stable Diffusion (PIL 로컬) → 실패 시 Mock URL

dalle-3 선택 시:
  DALL-E 3 API → 실패 시 Stable Diffusion → 실패 시 Mock URL
```
