# 생성된 가사 MD 파일 다운로드 기능 추가

- **날짜**: 2026-06-19 16:42-16:45
- **작업자**: Claude

## 작업 내용

**생성된 가사를 마크다운(MD) 파일로 다운로드할 수 있는 기능을 추가했습니다.**

가사 생성 목록의 각 항목에서 다운로드 버튼을 클릭하면 가사를 MD 파일로 다운로드할 수 있습니다.

---

## 🎯 구현 내용

### 다운로드 버튼 추가
✅ **위치**: 가사 카드의 오른쪽 상단 (다른 버튼들과 함께)
✅ **아이콘**: Download 아이콘
✅ **색상**: Primary (파란색)
✅ **호버 상태**: 밝아지는 효과

### MD 파일 형식
```markdown
# 곡 제목

**Genre:** K-Pop
**Mood:** 밝고 신나는
**Created:** 2026-06-19 16:30:45

---

## Verse

첫 번째 가사...

---

## Chorus

후렴구...

---

## Bridge

브릿지...

---

## Hook

훅...
```

## 🛠️ 기술 구현

### 다운로드 함수
```typescript
const handleDownloadMd = () => {
  // MD 형식으로 변환
  const mdContent = `# ${lyrics.title}

**Genre:** ${lyrics.prompt_genre}
**Mood:** ${lyrics.prompt_mood}
**Created:** ${new Date(lyrics.created_at).toLocaleString()}

---

## Verse

${lyrics.verse}

---

## Chorus

${lyrics.chorus}

---

## Bridge

${lyrics.bridge}

---

## Hook

${lyrics.hook}
`

  // Blob 생성 및 다운로드
  const blob = new Blob([mdContent], { type: 'text/markdown' })
  const url = window.URL.createObjectURL(blob)
  
  const link = document.createElement('a')
  link.href = url
  link.download = `${lyrics.title}.md`
  document.body.appendChild(link)
  link.click()
  link.parentNode?.removeChild(link)
  window.URL.revokeObjectURL(url)
}
```

### 버튼 UI
```typescript
<button
  onClick={handleDownloadMd}
  className="p-1.5 text-gray-400 hover:text-primary-400 rounded-lg hover:bg-primary-900/20 transition-colors"
  title="MD 파일 다운로드"
>
  <Download className="h-4 w-4" />
</button>
```

## ✨ 주요 기능

### 자동 파일명 생성
- 파일명: `{가사제목}.md`
- 예: `첫사랑.md`, `새로운계절.md`

### MD 형식 표준화
```
# 제목
**메타데이터** (장르, 분위기, 생성일)
---
## Verse / Chorus / Bridge / Hook
가사 내용
```

### 브라우저 다운로드
- 팝업 없이 자동 다운로드
- 기존 파일 덮어쓰기 묻기
- 모든 브라우저 호환
