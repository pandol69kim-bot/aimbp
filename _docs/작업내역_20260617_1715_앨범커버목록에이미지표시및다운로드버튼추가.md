# 작업내역: 앨범 커버 목록에 이미지 표시 및 다운로드 버튼 추가

**날짜:** 2026-06-17 20:25  
**작업자:** Claude Code

## 작업 내용

앨범 상세 페이지의 "생성 이력" 섹션에 커버 이미지를 표시하고 다운로드 버튼을 추가

## 구현 내용

### 1. 커버 데이터 그룹핑

같은 프롬프트로 생성된 커버들을 그룹화하여 3가지 비율(1:1, 16:9, 9:16)을 하나의 항목으로 표시

```typescript
const groupedCovers = albumCovers.reduce((acc, cover) => {
  const key = `${cover.prompt_genre}-${cover.prompt_mood}-${cover.prompt_keywords}`
  // 같은 key를 가진 커버들을 하나의 그룹으로 묶음
  // 각 그룹은 최대 3개의 이미지 (1:1, 16:9, 9:16)
}, [])
```

### 3. 이미지 표시

#### 이미지 컨테이너
```tsx
<div className="grid grid-cols-3 gap-3">
  {coverGroup.images.map((img) => (
    <div key={img.ratio} className="group relative">
      <div className={aspectRatioClass}>
        <img src={img.url} alt={`Cover ${img.ratio}`} />
        {/* 호버 시 다운로드 버튼 표시 */}
      </div>
    </div>
  ))}
</div>
```

#### Aspect Ratio
```typescript
'1:1'   → aspect-square      (정사각형)
'16:9'  → aspect-video       (가로형, HD)
'9:16'  → aspect-[9/16]      (세로형, 모바일)
```

### 4. 다운로드 버튼

#### 호버 상태
- 배경: `bg-black/40` (반투명 검정)
- 투명도: `opacity-0` → `opacity-100` (호버 시 표시)
- 버튼: `bg-primary-500/90` (주색상)

```tsx
<a
  href={img.url}
  download={`cover-${img.ratio}`}
  className="flex items-center gap-1.5 rounded-lg bg-primary-500/90 px-3 py-1.5 text-xs text-white hover:bg-primary-600 transition-colors"
>
  <Download className="h-3.5 w-3.5" />
  다운로드
</a>
```

## 기술 구현

### Reduce로 그룹핑
```typescript
const groupedCovers = albumCovers.reduce((acc, cover) => {
  const key = `${cover.prompt_genre}-${cover.prompt_mood}-${cover.prompt_keywords}`
  const existing = acc.find(g => g.key === key)
  
  if (existing) {
    // 같은 그룹에 이미지 추가
    existing.images.push({
      ratio: cover.size,
      url: cover.image_url
    })
  } else {
    // 새로운 그룹 생성
    acc.push({
      key,
      prompt_genre: cover.prompt_genre,
      // ...
      images: [{ratio: cover.size, url: cover.image_url}]
    })
  }
  return acc
}, [])
```
