## 작업 내용

**앨범 상세 페이지에 MP3 업로드 UI 탭을 추가했습니다.**

기존의 "기존 트랙 선택" 방식에서 "MP3 업로드"로 전환할 수 있는 탭 UI를 구현했습니다.

## 구현 완료 항목

### 1. ✅ 두 가지 모드 탭 추가
- **기존 트랙**: 이미 생성된 곡을 선택해서 앨범에 추가
- **MP3 업로드**: 사용자가 직접 MP3 파일을 업로드

### 2. ✅ MP3 업로드 폼
```
┌─────────────────────────────┐
│ [기존 트랙] [MP3 업로드]    │
├─────────────────────────────┤
│ MP3 파일 * (필수)           │
│ [파일 선택 버튼]             │
│                             │
│ 곡 제목 (선택)              │
│ [입력 필드]                  │
│                             │
│ 아티스트 (선택)             │
│ [입력 필드]                  │
│                             │
│ 장르 (선택)                 │
│ [입력 필드]                  │
│                             │
└─────────────────────────────┘
  [취소] [업로드]
```

### 3. ✅ 업로드 핸들러
- MP3 파일 검증
- 메타데이터 전송 (제목, 아티스트, 장르)
- 자동으로 앨범에 추가
- 업로드 진행 상태 표시

## 컴포넌트 사용

### Input 컴포넌트
```typescript
<Input
  type="file"
  label="MP3 파일 *"
  accept=".mp3"
  required
  onChange={(e) => setUploadFile(e.target.files?.[0] || null)}
/>
```

### 탭 버튼
```typescript
<button
  onClick={() => setAddTrackMode('upload')}
  className={`px-3 py-2 text-sm font-medium transition-colors ${
    addTrackMode === 'upload'
      ? 'text-primary-400 border-b-2 border-primary-400'
      : 'text-gray-400 hover:text-gray-300'
  }`}
>
  MP3 업로드

```
