## 생성되는 이미지 포맷

- **포맷**: Data URL (Base64 encoded PNG)
- **크기별 해상도**:
  - 1:1 → 512×512px
  - 16:9 → 1024×576px
  - 9:16 → 576×1024px

- **분위기별 색상**:
  - peaceful: 파란색 그래디언트
  - dark: 검은색 그래디언트
  - energetic: 주황색 그래디언트
  - romantic: 분홍색 그래디언트
  - melancholic: 보라색 그래디언트
