## 작업 내용

트랙 생성 목록에서 생성 실패한 트랙의 실패 사유를 상세히 볼 수 있는 기능을 추가했습니다.

### 주요 개선사항

1. **실패 상태 시각화 강화**
   - 실패한 트랙의 카드 배경을 빨강색으로 강조
   - 테두리도 빨강색으로 구분
   - 시각적으로 문제 상황을 명확하게 표현

2. **접기/펴기 가능한 상세 정보**
   - "생성 실패" 헤더와 화살표 아이콘
   - 클릭하면 상세 사유가 펼쳐짐
   - 화살표가 회전하며 상태를 시각적으로 표현

3. **상세 에러 정보 박스**
   - 빨강 배경의 구분된 박스에 표시
   - 모노스페이스 폰트로 기술적 정보 강조
   - 텍스트 줄바꿈으로 긴 메시지 대응

4. **사용자 친화적 도움말**
   - "다른 AI 서비스로 재시도하거나 설정을 변경해보세요"
   - 💡 이모지로 시각적 강조
   - 사용자가 다음 행동을 알 수 있도록 안내

## 기술 구현 상세

### 1. State 관리
```typescript

```

### 2. 조건부 렌더링
```typescript
const isFailed = displayTrack.status === 'failed'
const hasError = isFailed && displayTrack.error_message
```

### 3. 접기/펴기 버튼
```tsx
<button
  onClick={() => setShowError(!showError)}
  className="flex items-start gap-2 flex-1 text-left hover:opacity-80 transition-opacity"
>
  <AlertCircle className="h-4 w-4 text-red-400 flex-shrink-0 mt-0.5" />
  <div>
    <p className="text-xs font-medium text-red-400">생성 실패</p>
    <p className="text-xs text-red-300/80 truncate">
      {showError ? displayTrack.error_message : '사유 보기'}
    </p>
  </div>
  <ChevronDown className={`... ${showError ? 'rotate-180' : ''}`} />
</button>
```

### 4. 상세 정보 박스
