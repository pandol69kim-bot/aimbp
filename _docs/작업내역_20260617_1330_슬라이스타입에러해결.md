# 작업내역: TypeError - tracks.slice is not a function 해결

**날짜:** 2026-06-17 13:30  
**작업자:** Claude Code

## 에러

```
TypeError: tracks.slice is not a function
src/app/(dashboard)/dashboard/page.tsx (25:32)
```

## 원인

API 응답 형식 불일치:

Backend 응답 형식:
```json
{
  "success": true,
  "data": [ { track }, { track } ],
  "error": null
}
```

Frontend 훅 예상:
```typescript
const response = await api.get('/music')
return response.data  // ← 전체 응답 객체 { success, data: [...], error }
```

실제 배열은 `response.data.data` 에 있음

## 해결 방법

모든 LIST 훅 수정 (`response.data` → `response.data.data`):

### useTrackList (useLyrics.ts)
```typescript
const response = await api.get('/music')
return response.data.data || []  // ← 배열 반환
```

### useLyricsList (useMusic.ts)
```typescript
const response = await api.get('/lyrics')
return response.data.data || []  // ← 배열 반환
```

### useAlbumList (useAlbums.ts)
```typescript
const response = await api.get('/albums')
return response.data.data || []  // ← 배열 반환
```

## 수정 파일

- `frontend/src/hooks/useMusic.ts`: useTrackList
- `frontend/src/hooks/useLyrics.ts`: useLyricsList
- `frontend/src/hooks/useAlbums.ts`: useAlbumList
