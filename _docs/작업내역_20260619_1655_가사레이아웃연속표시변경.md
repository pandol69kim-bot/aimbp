## 작업 내용

**가사 표시 레이아웃을 분리된 칸(그리드)에서 한 화면에 연속으로 표시하도록 변경했습니다.**

사용자가 가사를 읽을 때 마치 전체 곡을 읽는 것처럼 자연스러운 흐름으로 볼 수 있습니다.

---

## 🔄 변경 내용

### Before (분리된 칸)
```
┌─────────────────────────────────┐
│ VERSE           │ CHORUS         │
│ 가사1           │ 가사2          │
│                 │                │
├─────────────────────────────────┤
│ BRIDGE          │ HOOK           │
│ 가사3           │ 가사4          │
│                 │                │
└─────────────────────────────────┘
```

**문제:**
- 2x2 그리드로 분리됨
- 각각 다른 박스에 표시
- 읽는 흐름이 끊김
- 화면 스크롤 필요

### After (연속 표시)
```
┌─────────────────────────────────┐
│ VERSE                           │
│ 가사1                           │
│ (가사 내용...)                  │
│                                 │
│ CHORUS                          │
│ 가사2                           │
│ (가사 내용...)                  │
│                                 │
│ BRIDGE                          │
│ 가사3                           │
│ (가사 내용...)                  │
│                                 │
│ HOOK                            │
│ 가사4                           │
│ (가사 내용...)                  │
└─────────────────────────────────┘
```

**개선:**
- 한 열로 연속 표시
- 자연스러운 읽기 흐름
- 전체 가사를 한 화면에서 볼 수 있음
- 마치 악보를 읽는 느낌

## 📐 UI 개선

### 섹션 간 간격
```
space-y-6  →  6 * 0.25rem = 1.5rem 간격
(Before: gap-4 = 4 * 0.25rem = 1rem)
```

### 줄 간격
```
leading-relaxed  →  1.625 배 줄 높이
(더 읽기 편함)
```

### 텍스트 색상
```
text-gray-300 → text-gray-200
(더 밝고 명확함)
```
