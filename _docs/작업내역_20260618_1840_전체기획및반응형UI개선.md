## 주요 결정 사항

### 반응형 기준점
- 모바일: < 768px (슬라이드인 사이드바)
- 데스크톱: >= 768px (고정 사이드바)

### 상태 관리
- Zustand를 사용한 글로벌 사이드바 상태
- 모바일에서만 상태 변화 활용

### 애니메이션
- transform 기반 (성능 최적화)
- duration-300 ease-in-out
- lg:translate-x-0로 데스크톱에서 비활성화

### 문서화 전략
- 기획서: 새 프로젝트 참고용
- 프로세스: 기능별 상세 정의서
- 기술 스택 명시, 구현 코드 포함

## 기술 스택 및 패턴

### 상태 관리
```typescript
// Zustand를 활용한 단순하고 효율적인 상태 관리
const { isOpen, toggleSidebar } = useSidebarStore()
```

### Tailwind 반응형 클래스
```
lg:hidden          - 데스크톱에서 숨김
lg:static          - 데스크톱에서 정적 위치
lg:translate-x-0   - 데스크톱에서 애니메이션 무시
hidden sm:block    - 모바일에서 숨김, 태블릿 이상에서 표시
```
