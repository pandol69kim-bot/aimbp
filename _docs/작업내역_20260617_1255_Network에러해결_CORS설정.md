# 작업내역: Network Error 해결 - CORS 설정 수정

**날짜:** 2026-06-17 12:55  
**작업자:** Claude Code

## 문제 분석

Frontend 로그인 시 "Network Error" 발생
- 원인: CORS 설정 미스매치

## 근본 원인

| 설정 | 값 | 상태 |
|------|-----|------|
| Backend CORS_ORIGINS | http://localhost:3000 | ❌ 3001 미포함 |
| Frontend 포트 | 3001 | ❌ CORS에서 거부 |

## 해결 방법

### 1. .env 파일 수정

```env
# Before
CORS_ORIGINS=http://localhost:3000

# After
CORS_ORIGINS=http://localhost:3000,http://localhost:3001
```

### 2. docker-compose.yml 수정

```yaml
# Before
NEXT_PUBLIC_API_URL=/api/v1

# After
NEXT_PUBLIC_API_URL=http://localhost:8001/api/v1
```

### 3. next.config.js 유지

상대 경로 rewrites 제거 → 직접 API 호출로 변경

## 최종 구성

```
브라우저 (localhost:3001)
  ↓
Frontend (Next.js) 
  ↓
axios baseURL: http://localhost:8001/api/v1
  ↓
Backend (localhost:8001) - CORS 허용
  ↓
API 응답
```
