# 가사 멀티 드래그드롭 업로드 기능

- **날짜**: 2026-06-19 16:46-16:50
- **작업자**: Claude
- **소요시간**: 4분

## 작업 내용

**여러 MD 파일을 한 번에 드래그드롭으로 업로드해서 가사 목록에 자동으로 추가되도록 개선했습니다.**

기존의 단일 파일 업로드에서 멀티 파일 업로드로 확장되어, 사용자가 여러 가사를 한 번에 생성 목록에 추가할 수 있습니다.

---

## 🎯 구현 내용

### Before (단일 파일)
```
MD 파일 업로드
└─ 1개 파일만 선택 가능
   └─ [파일 업로드] 클릭
       └─ 1개씩 추가
```

### After (멀티 파일)
```
MD 파일 업로드
├─ 여러 파일 동시 드래그 드롭
├─ 여러 파일 동시 선택
└─ 일괄 업로드
   ├─ 파일별 진행도 표시
   └─ 모두 추가 완료
```

### 3. 드래그드롭 핸들러
```typescript
const handleDrop = (e: React.DragEvent) => {
  const files = Array.from(e.dataTransfer.files).filter((file) =>
    file.name.toLowerCase().endsWith('.md')
  )
  
  // Before: setUploadFile(files[0])
  // After: 여러 파일 추가
  if (files.length > 0) {
    setUploadFiles((prev) => [...prev, ...files])  // ✅ 누적
  }
}
```

### 4. 파일 선택 핸들러
```typescript
const handleFileSelect = (e: React.ChangeEvent<HTMLInputElement>) => {
  if (e.target.files) {
    const files = Array.from(e.target.files)
    // 기존 파일에 추가
    setUploadFiles((prev) => [...prev, ...files])
  }
}
```

### 5. 파일 제거 함수
```typescript
const removeFile = (index: number) => {
  setUploadFiles((prev) => prev.filter((_, i) => i !== index))
}
```

### 6. 멀티 파일 업로드
```typescript
const handleUploadMd = async () => {
  setIsUploading(true)
  let uploadedCount = 0

  for (let i = 0; i < uploadFiles.length; i++) {
    const file = uploadFiles[i]
    const fileKey = file.name

    try {
      // 진행도 0으로 시작
      setUploadProgress((prev) => ({
        ...prev,
        [fileKey]: 0,
      }))

      // 파일 업로드
      const response = await api.post('/lyrics/upload', formData)
      
      // 진행도 100%
      setUploadProgress((prev) => ({
        ...prev,
        [fileKey]: 100,
      }))

      uploadedCount++
    } catch (error) {
      // 실패: -1 표시
      setUploadProgress((prev) => ({
        ...prev,
        [fileKey]: -1,
      }))
    }
  }

  alert(`${uploadedCount}개의 가사가 생성 목록에 추가되었습니다.`)
}
```
