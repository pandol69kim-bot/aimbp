# AI Music Business Platform — 개발계획서

**작성일:** 2026-06-17
**기반 문서:** 기획 ㅡ20260616ㅡAI Music Business Platform.md

---

## 1. 프로젝트 개요

AI 도구를 활용하여 작사·작곡·편곡·음원 제작·앨범 발매·글로벌 유통·저작권 등록·수익 정산까지 올인원으로 처리하는 음악 비즈니스 플랫폼.

---

## 2. 기술 스택

| 영역 | 기술 |
|------|------|
| Frontend | Next.js, React, Tailwind CSS |
| Backend | FastAPI (Python) |
| Database | PostgreSQL |
| Storage | AWS S3 |
| AI | OpenAI API, Claude API, Gemini API, Suno API, Udio API |
| Infra | AWS, Docker, Kubernetes |

---

## 3. 시스템 아키텍처

```
[Next.js Frontend]
        ↓
[FastAPI Backend]
   ↙     ↓     ↘
[PostgreSQL] [S3] [AI Services]
                  ↙   ↓    ↘
              OpenAI Claude Suno/Udio
```

---

## 4. 개발 단계 (Phase)

### Phase 1 — 핵심 기반 (MVP)
**목표:** 음악 생성 → 저장 → 다운로드 기본 플로우 완성

| 항목 | 내용 |
|------|------|
| 인증 | 회원가입 / 로그인 / JWT |
| AI 작사 | GPT/Claude 연동, Verse·Chorus·Bridge 출력 |
| AI 작곡 | Suno API / Udio API 연동, MP3·WAV 출력 |
| 파일 관리 | S3 업로드/다운로드 |
| 기본 대시보드 | 생성 이력 목록 |

**예상 기간:** 6주

---

### Phase 2 — 음악 제작 풀파이프라인
**목표:** 편곡·보컬·마스터링까지 AI 자동화

| 항목 | 내용 |
|------|------|
| AI 편곡 | 드럼·베이스·기타·피아노 등 자동 생성, STEM 출력 |
| AI 보컬 | 성별·음색 선택, 다국어 지원 |
| AI 마스터링 | Loudness·EQ·Stereo Imaging 자동화, 플랫폼별 최적화 |
| 앨범 커버 생성 | GPT Image / Flux / Stable Diffusion 연동 |

**예상 기간:** 6주

---

### Phase 3 — 저작권 & 유통
**목표:** 저작권 관리 및 글로벌 유통 연동

| 항목 | 내용 |
|------|------|
| 저작권 등록 | 생성 로그·프롬프트·버전 이력 저장 |
| 블록체인 옵션 | 해시 생성, NFT 등록, 원본 증명 |
| 글로벌 유통 | Spotify·Apple Music·YouTube Music·Amazon 등 |
| 국내 유통 | Melon·Genie·FLO·Bugs·VIBE |
| 발매 스케줄링 | 예약 발매, 국가별·시간대 설정 |

**예상 기간:** 8주

---

### Phase 4 — 수익 & 마케팅
**목표:** 수익 분석 대시보드 및 AI 마케팅 자동화

| 항목 | 내용 |
|------|------|
| 수익 대시보드 | 총 수익, 일/월별, 국가별, 플랫폼별 |
| 저작권료 정산 | 저작권료·인접권·YouTube Content ID |
| AI 마케팅 | 유튜브 제목·설명·태그·쇼츠/릴스 스크립트 생성 |
| SNS 자동 배포 | Instagram, TikTok, YouTube, X, Facebook |
| 유튜브 수익화 | Lyric Video·Visualizer·Shorts 자동 생성 및 업로드 |

**예상 기간:** 6주

---

### Phase 5 — 구독 모델 & 관리자
**목표:** 수익화 모델 적용 및 운영 도구 완성

| 항목 | 내용 |
|------|------|
| 구독 플랜 | Free(5곡/월), Creator(100곡), Studio(무제한), Label(다중계정) |
| 결제 연동 | 구독 결제, 출금 처리 |
| 관리자 패널 | 사용자·정산·콘텐츠 검수 |
| AI 생성 검증 | 저작권 검사, AI 생성 여부, 정책 위반 감지 |

**예상 기간:** 4주

## 5. 핵심 KPI

| 지표 | 목표 |
|------|------|
| 음원 생성 수 | 월 100만 곡 |
| 발매 수 | 월 10만 곡 |

| 월 정산 규모 | 100억 원 |
| MRR | 10억 원 이상 |

## 6. 우선순위 API 연동 목록

| 서비스 | 용도 | 우선순위 |
|--------|------|----------|
| OpenAI API | 작사, 마케팅 콘텐츠 생성 | 최상 |
| Claude API | 작사 보조, 가사 품질 향상 | 최상 |
| Suno API | AI 작곡·음원 생성 | 최상 |
| Udio API | AI 작곡 대안 | 상 |
| Stable Diffusion / Flux | 앨범 커버 생성 | 중 |
| Runway / Kling | 뮤직비디오·쇼츠 영상 생성 | 중 |
| Gemini API | 다국어 작사, 분석 | 중 |

## 9. 최종 비전

> **"1인 AI 뮤직 레이블"**
>
> 누구나 AI를 이용하여 작사 → 작곡 → 편곡 → 음원 제작 → 발매 → 유통 → 정산 → 수익화까지 수행하고,
> 모든 저작권과 스트리밍 수익을 직접 소유하는 글로벌 음악 비즈니스 플랫폼.
