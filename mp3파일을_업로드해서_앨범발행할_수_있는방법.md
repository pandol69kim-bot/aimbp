## 시작 전 준비

### 필요한 파일들

앨범 발행을 위해 다음 파일들이 준비되어야 합니다:

| 항목 | 요구사항 | 형식 | 크기 제한 |
|------|---------|------|---------|
| **MP3 파일** | ✅ 필수 | .mp3 | 최대 500MB/파일 |
| **커버아트** | ✅ 필수 | .jpg, .png | 최대 5MB, 최소 400x400px |
| **앨범 제목** | ✅ 필수 | 텍스트 | 최대 100자 |
| **아티스트명** | ✅ 필수 | 텍스트 | 최대 100자 |
| **발매일** | ✅ 필수 | 날짜 | YYYY-MM-DD |

### 시작하기

1. AIMBP 플랫폼에 로그인합니다
2. 대시보드의 **"앨범"** 메뉴로 이동합니다
3. **"새 앨범 만들기"** 버튼을 클릭합니다

## 앨범 만들기

### Step 1: 앨범 기본 정보 입력

![앨범 생성 화면]

새 앨범을 만들기 위해 다음 정보를 입력합니다:

```
┌─────────────────────────────────┐
│ 새 앨범 만들기                   │
├─────────────────────────────────┤
│ 앨범 제목 *                      │
│ [앨범 이름을 입력하세요]          │
│                                  │
│ 아티스트명 *                     │
│ [아티스트 또는 그룹명]           │
│                                  │
│ 발매일 *                        │
│ [YYYY-MM-DD]                    │
│                                  │
│ 장르 (선택사항)                  │
│ [Pop / Rock / Jazz / ...]       │
│                                  │
│ 설명 (선택사항)                  │
│ [앨범에 대한 설명을 입력...]     │
│                                  │
│ [만들기] [취소]                  │
└─────────────────────────────────┘
```

**예시:**
- 앨범 제목: "Summer Vibes"
- 아티스트명: "John Doe"
- 발매일: 2026-07-15
- 장르: Pop

### Step 2: 앨범 생성 완료

앨범이 **초안(Draft)** 상태로 생성됩니다.

```
✅ 앨범 "Summer Vibes"가 생성되었습니다!
상태: 초안 (아직 수정 가능)
```

## MP3 파일 업로드

### Step 1: 트랙 추가 버튼 클릭

앨범 상세 페이지에서 **"음원 추가"** 또는 **"+ 트랙 추가"** 버튼을 클릭합니다.

### Step 2: MP3 파일 선택

```
┌─────────────────────────────────┐
│ 음원 파일 업로드                 │
├─────────────────────────────────┤
│ [파일 선택] 또는 드래그 & 드롭   │
│                                  │
│ 지원 형식: .mp3                 │
│ 최대 크기: 500MB                │
│                                  │
│ [업로드 시작]                    │
└─────────────────────────────────┘
```

**옵션:**
- **파일 선택 버튼**: 클릭해서 컴퓨터에서 파일 선택
- **드래그 & 드롭**: MP3 파일을 영역에 드래그해서 놓기

### Step 3: 업로드 진행 상황 확인

```
파일: "song-1.mp3" (45.2 MB)
진행률: ████████████░░░░░░░ 65%
속도: 2.3 MB/s
예상 시간: 15초
```

### 여러 파일 동시 업로드

최대 **5개 파일**을 동시에 업로드할 수 있습니다:

1. 첫 번째 파일 업로드 시작
2. 업로드 중에 두 번째 파일 선택
3. 업로드 진행 중에 세 번째 파일 선택
4. 모든 파일이 동시에 업로드됨

## 트랙 정보 입력

### Step 1: 업로드된 트랙 정보 수정

각 업로드된 MP3 파일에 대해 트랙 정보를 입력합니다:

```
┌──────────────────────────────────┐
│ 트랙 정보 입력                    │
├──────────────────────────────────┤
│ 트랙 번호: 1                      │
│                                   │
│ 트랙 제목 *                      │
│ [예: Summer Dreams]               │
│                                   │
│ 아티스트 *                       │
│ [예: John Doe]                   │
│                                   │
│ 작곡가 (선택사항)                │
│ [작곡가명]                        │
│                                   │
│ 작사가 (선택사항)                │
│ [작사가명]                        │
│                                   │
│ 가사 (선택사항)                  │
│ [가사 텍스트]                    │
│                                   │
│ 설명 (선택사항)                  │
│ [이 트랙에 대한 설명]            │
│                                   │
│ [저장] [삭제] [취소]             │
└──────────────────────────────────┘
```

**필수 입력 항목:**
- ✅ 트랙 제목: 곡의 이름
- ✅ 아티스트: 곡의 가수/연주자

**선택 입력 항목:**
- 작곡가: 작곡한 사람
- 작사가: 가사를 쓴 사람
- 가사: 곡의 가사 텍스트
- 설명: 곡에 대한 추가 설명

### Step 2: 각 트랙별로 정보 입력

앨범의 모든 트랙에 대해 위 과정을 반복합니다.

**예시:**

| 트랙 | 제목 | 아티스트 | 길이 | 상태 |
|------|------|---------|------|------|
| 1 | Summer Dreams | John Doe | 3:45 | ✅ 완료 |
| 2 | Sunset | John Doe | 4:20 | ✅ 완료 |
| 3 | Moonlight | John Doe | 3:30 | ✅ 완료 |

│                                   │

│ 레이블 (선택사항)                │

```

```

## 앨범 발행하기

### Step 1: 발행 전 확인 사항

모든 필수 정보가 입력되었는지 확인합니다:

```
발행 준비 확인:

✅ 앨범 제목: "Summer Vibes"
✅ 아티스트: "John Doe"  
✅ 발매일: 2026-07-15
✅ 커버아트: 업로드됨
✅ 음원 파일: 3개 (Summer Dreams, Sunset, Moonlight)
✅ 앨범 완성도: 100%

모든 항목이 준비되었습니다! 발행할 수 있습니다.
```

### Step 2: "발행하기" 버튼 클릭

앨범 상세 페이지의 **"발행하기"** 버튼을 클릭합니다.

### Step 3: 발행 확인 대화상자

```
┌──────────────────────────────────┐
│ 앨범 발행 확인                    │
├──────────────────────────────────┤
│                                   │
│ 앨범: "Summer Vibes"를            │
│ 정식으로 발행하시겠습니까?       │
│                                   │
│ 발행 후:                         │
│ ⚠️  앨범 정보 수정 불가능        │
│ ⚠️  음원 추가/삭제 불가능        │
│ ✅ 다운로드 가능                 │
│                                   │
│ [발행하기] [취소]                │
└──────────────────────────────────┘
```

**주의:** 발행한 앨범은 정보를 수정할 수 없습니다. 발행 전에 모든 정보를 다시 확인하세요.

### Step 4: 발행 완료

```
✅ 앨범이 성공적으로 발행되었습니다!

앨범: Summer Vibes
상태: 🔒 발행됨 (잠금)
발행일시: 2026-06-19 14:30

이제 사용자들이 이 앨범을 다운로드할 수 있습니다.
```

### 앨범 다운로드

발행된 앨범은 다음과 같이 다운로드할 수 있습니다:

#### 1. 개별 음원 다운로드

각 트랙 옆의 **⬇ 다운로드** 버튼을 클릭:

```
Summer Dreams (3:45) [▶ 듣기] [⬇ 다운로드]
↓
파일: Summer Dreams.mp3 (3.5 MB) 다운로드 시작
```

#### 2. 앨범 전체 다운로드 (ZIP)

**"⬇ 앨범 전체 다운로드"** 버튼 클릭:

```
┌──────────────────────────────────┐
│ 앨범 다운로드                     │
├──────────────────────────────────┤
│ 앨범: Summer Vibes              │
│ 트랙 수: 3개                     │
│ 예상 용량: 45.2 MB              │
│                                   │
│ 다운로드 형식:                   │
│ ○ ZIP 파일 (권장) - 모든 파일 포함│
│ ○ 개별 다운로드 - 각 트랙 개별   │
│                                   │
│ [다운로드 시작] [취소]           │
└──────────────────────────────────┘

⏳ 다운로드 진행 중...

파일: summer-vibes.zip
진행률: ████████████░░░░░░░░░░░░░░ 60%
예상 시간: 1분 30초

다운로드가 완료되면 폴더 구조:
summer-vibes/

│   ├── 01. Summer Dreams.mp3
│   ├── 02. Sunset.mp3

```

## 자주 묻는 질문 (FAQ)

### Q1. MP3 파일 업로드 시 오류가 발생합니다. 어떻게 해야 하나요?

**A.** 다음을 확인하세요:

- ✅ 파일 형식이 `.mp3`인지 확인
- ✅ 파일 크기가 500MB 이하인지 확인
- ✅ 네트워크 연결 상태 확인
- ✅ 다른 브라우저 시도 (Chrome, Firefox 등)
- ✅ 캐시 삭제 후 재시도

**오류 메시지별 해결법:**

| 오류 | 원인 | 해결법 |
|------|------|--------|
| File format not supported | MP3 파일이 아님 | MP3로 변환 후 재업로드 |
| File size exceeds limit | 파일이 너무 큼 | 파일 압축 또는 분할 |
| Network error | 네트워크 끊김 | 인터넷 연결 확인 후 재시도 |

### Q2. 발행한 앨범을 수정할 수 없다고 합니다.

**A.** 발행된 앨범은 **수정이 불가능**하도록 잠금됩니다. 이는 버전 관리와 데이터 무결성을 보장하기 위함입니다.

수정이 필요한 경우:
1. 관리자에게 **발행 취소** 요청
2. 앨범 정보 수정
3. 다시 발행

### Q3. 커버아트 크기가 부족하다는 메시지가 나옵니다.

**A.** 커버아트 최소 크기는 **400x400px**입니다.

**해결법:**
1. 이미지 편집 프로그램 (Photoshop, GIMP 등)으로 이미지 크기 조정
2. 온라인 리사이저 사용: https://resizeimage.net/
3. 최소 400x400px, 권장 1000x1000px 이상의 이미지 준비

### Q4. 같은 앨범에서 다양한 아티스트 곡을 포함할 수 있나요?

**A.** 네, 가능합니다! 각 트랙마다 다른 아티스트를 지정할 수 있습니다.

```
트랙 1: Summer Dreams - John Doe
트랙 2: Sunset - Jane Smith
트랙 3: Moonlight - John Doe & Jane Smith
```

### Q5. 발행된 앨범을 삭제할 수 있나요?

**A.** 발행된 앨범은 **삭제할 수 없습니다**. 데이터 무결성을 보장하기 위해 의도적으로 제한됩니다.

필요시 관리자에게 문의하세요.

### Q6. 다운로드 파일이 손상되었을 가능성이 있습니다.

**A.** 다음을 시도하세요:

1. 다시 다운로드
2. 다른 MP3 플레이어로 재생 시도
3. ZIP 파일이 손상된 경우 압축 풀기 도구 확인
4. 관리자에게 문의

### Q7. 업로드 중에 음원 정보를 입력할 수 있나요?

**A.** 아니요, 파일이 완전히 업로드된 후에 정보를 입력해야 합니다.

**순서:**
1. MP3 파일 업로드 완료
2. 진행 상황 표시: 100%
3. 트랙 정보 입력 (제목, 아티스트 등)
4. 저장

### Q8. 앨범 발행에 비용이 드나요?

**A.** 현재는 **무료**입니다. 추후 프리미엄 기능 추가 시 공지될 예정입니다.

### Q9. 발행된 앨범은 몇 명이 다운로드할 수 있나요?

**A.** **제한이 없습니다**. 무제한으로 다운로드 가능합니다.

다운로드 통계는 **대시보드**에서 확인할 수 있습니다:
- 총 다운로드 수
- 월별 다운로드 수
- 각 트랙별 다운로드 수

### Q10. 앨범 발행 후 설명 텍스트만 추가할 수 있나요?

**A.** 아니요, 발행된 앨범의 **모든 정보는 수정 불가능**합니다.

이는 다음을 보장하기 위함입니다:
- 음악 무결성
- 버전 관리
- 라이선스 보호
