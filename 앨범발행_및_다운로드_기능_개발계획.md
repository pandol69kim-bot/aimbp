## 1. 개요

### 1.1 목표

AIMBP 플랫폼에서 제작한 앨범을 정식으로 발행(배포)한 후, **발행된 앨범은 수정 불가능하게 잠금**하고 동시에 **사용자가 발행된 앨범을 다운로드**할 수 있는 기능 구현.

### 1.2 핵심 기능

| # | 기능 | 설명 | 상태 |
|---|------|------|------|
| 1 | 앨범 발행 | Draft → Published 상태 전환 | 📝 필수 |
| 2 | 수정 금지 | 발행된 앨범 수정 차단 | 📝 필수 |
| 3 | 다운로드 | 발행된 앨범 전체 다운로드 (ZIP) | 📝 필수 |
| 4 | 음원 개별 다운로드 | 각 트랙 개별 다운로드 | 📝 필수 |
| 5 | 발행 취소 | Published → Draft 상태 복구 | 🟠 선택 |
| 6 | 버전 관리 | 앨범 발행 이력 추적 | 🟠 선택 |
| 7 | 다운로드 이력 | 사용자별 다운로드 기록 | 🟡 향후 |
| 8 | 다운로드 제한 | 구독 플랜별 다운로드 횟수 제한 | 🟡 향후 |

## 3. 앨범 상태 관리

### 3.1 앨범 상태 정의

```
DRAFT (초안)
  └─ 사용자가 생성했지만 아직 발행하지 않은 상태
  └─ 모든 수정 가능
  └─ 다운로드 불가능
  └─ 음원 추가/삭제/수정 가능
  └─ 생성자 외 접근 불가

        ↓ [발행하기]

PUBLISHED (발행됨)
  └─ 정식으로 배포된 상태
  └─ 수정/삭제 불가능 (잠금)
  └─ 다운로드 가능
  └─ 읽기 전용
  └─ (선택) 발행 취소로 DRAFT로 복구

        ↓ [발행 취소] (선택)

ARCHIVED (보관됨, 선택사항)
  └─ 더 이상 활성화되지 않은 상태
  └─ 읽기 전용
  └─ 다운로드 가능 (이력 조회만)
```

### 3.2 상태 전환 다이어그램

```
┌─────────┐
│ DRAFT   │
└────┬────┘
     │ [Publish]
     ▼
┌─────────────────┐
│ PUBLISHED       │ ◄─────── [Unpublish] (선택)
├─────────────────┤
│ - ReadOnly      │
│ - Downloadable  │
│ - ChangesFrozen │
└────────┬────────┘
         │ [Archive] (향후)
         ▼
    ┌─────────┐
    │ ARCHIVED│
    └─────────┘
```

## 4. 발행 기능 (Publish)

### 4.1 발행 전 검증

```python
class AlbumPublishValidator:
    async def validate(self, album: Album) -> ValidationResult:
        errors = []
        
        # 1. 기본 정보 확인
        if not album.title:
            errors.append("Album title is required")
        if not album.artist:
            errors.append("Artist name is required")
        if not album.release_date:
            errors.append("Release date is required")
        
        # 2. 커버아트 확인
        if not album.cover_url:
            errors.append("Album cover is required")
        
        # 3. 음원 확인
        if not album.tracks or len(album.tracks) == 0:
            errors.append("Album must have at least one track")
        
        # 4. 각 음원 유효성 검증
        for i, track in enumerate(album.tracks):
            if not track.title:
                errors.append(f"Track {i+1}: title is required")
            if not track.music_file_url:
                errors.append(f"Track {i+1}: audio file is missing")
            if not track.duration:
                errors.append(f"Track {i+1}: duration is required")
        
        # 5. 필수 정보 완성도 확인
        completion = self._calculate_completion(album)
        if completion < 100:
            errors.append(f"Album is {100-completion}% incomplete")
        
        return ValidationResult(
            is_valid=len(errors) == 0,
            errors=errors
        )
```

### 4.2 발행 프로세스

```python
class AlbumPublishService:
    async def publish_album(self, album_id: str, user_id: str) -> Album:
        """앨범을 발행하고 상태 변경"""
        
        # 1. 앨범 조회 및 소유권 확인
        album = await db.get(Album, album_id)
        if album.user_id != user_id:
            raise PermissionError("Not album owner")
        
        # 2. 현재 상태 확인
        if album.status != AlbumStatus.DRAFT:
            raise ValueError(f"Album is already {album.status}")
        
        # 3. 발행 전 검증
        validator = AlbumPublishValidator()
        validation = await validator.validate(album)
        if not validation.is_valid:
            raise ValidationError(validation.errors)
        
        # 4. 상태 업데이트
        album.status = AlbumStatus.PUBLISHED
        album.published_at = datetime.utcnow()
        album.is_locked = True
        
        # 5. 변경 불가능 플래그 설정
        album.is_immutable = True
        
        # 6. 데이터베이스에 저장
        await db.commit()
        
        # 7. 발행 이벤트 발송
        await event_bus.emit(
            event=AlbumPublishedEvent(
                album_id=album.id,
                user_id=user.id,
                published_at=album.published_at
            )
        )
        
        return album

    async def unpublish_album(self, album_id: str, user_id: str) -> Album:
        """발행 취소 (선택사항)"""
        
        album = await db.get(Album, album_id)
        if album.user_id != user_id:
            raise PermissionError("Not album owner")
        
        if album.status != AlbumStatus.PUBLISHED:
            raise ValueError(f"Cannot unpublish {album.status} album")
        
        # 경고: 다운로드 이력이 있으면 취소 불가
        download_count = await self._count_downloads(album.id)
        if download_count > 0:
            raise ValueError(
                f"Cannot unpublish album with {download_count} downloads"
            )
        
        album.status = AlbumStatus.DRAFT
        album.is_locked = False
        album.is_immutable = False
        
        await db.commit()
        return album
```

## 5. 다운로드 기능

### 5.1 앨범 전체 다운로드 (ZIP)

#### 구현 방식

```python
# services/album_download_service.py
from zipfile import ZipFile
import io
import asyncio

class AlbumDownloadService:
    
    async def create_album_zip(self, album_id: str) -> bytes:
        """앨범을 ZIP 파일로 압축 (메모리 기반)"""
        
        album = await db.get(Album, album_id)
        if album.status != AlbumStatus.PUBLISHED:
            raise ValueError("Only published albums can be downloaded")
        
        # 메모리 기반 ZIP 생성
        zip_buffer = io.BytesIO()
        
        with ZipFile(zip_buffer, 'w', compression=ZIP_DEFLATED) as zip_file:
            # 1. 커버 아트 추가
            if album.cover_url:
                cover_data = await self._download_file(album.cover_url)
                zip_file.writestr(
                    f"{album.slug}/cover.jpg",
                    cover_data
                )
            
            # 2. 모든 음원 추가
            audio_dir = f"{album.slug}/audio"
            for i, track in enumerate(album.tracks, 1):
                audio_data = await self._download_file(track.music_file_url)
                filename = f"{i:02d}. {track.title}.mp3"
                zip_file.writestr(
                    f"{audio_dir}/{filename}",
                    audio_data
                )
            
            # 3. 메타데이터 JSON 추가
            metadata = {
                "title": album.title,
                "artist": album.artist,
                "release_date": album.release_date.isoformat(),
                "genre": album.genre,
                "total_tracks": len(album.tracks),
                "duration": sum(t.duration for t in album.tracks),
                "tracks": [
                    {
                        "number": i,
                        "title": track.title,
                        "duration": track.duration,
                        "artists": [track.artist]
                    }
                    for i, track in enumerate(album.tracks, 1)
                ]
            }
            
            import json
            zip_file.writestr(
                f"{album.slug}/metadata.json",
                json.dumps(metadata, indent=2, ensure_ascii=False)
            )
        
        zip_buffer.seek(0)
        return zip_buffer.getvalue()

    async def initiate_download(
        self,
        album_id: str,
        user_id: str
    ) -> DownloadToken:
        """다운로드 요청 시작 (큰 파일은 비동기 처리)"""
        
        album = await db.get(Album, album_id)
        
        # 1. 접근 권한 확인
        if not self._can_download(album, user_id):
            raise PermissionError("Not authorized to download")
        
        # 2. 총 용량 계산
        total_size = await self._calculate_total_size(album)
        
        # 3. 크기에 따라 처리 방식 결정
        if total_size < 500 * 1024 * 1024:  # 500MB 이상
            # 동기 처리 (즉시 반환)
            zip_data = await self.create_album_zip(album_id)
            
            download_token = DownloadToken(
                id=uuid4(),
                album_id=album_id,
                user_id=user_id,
                file_data=zip_data,
                file_size=len(zip_data),
                expires_at=datetime.utcnow() + timedelta(hours=24),
                status="ready"
            )
        else:
            # 비동기 처리 (배경 작업)
            download_token = DownloadToken(
                id=uuid4(),
                album_id=album_id,
                user_id=user_id,
                expires_at=datetime.utcnow() + timedelta(hours=24),
                status="preparing"
            )
            
            # 백그라운드 작업 시작
            asyncio.create_task(
                self._prepare_download_async(download_token)
            )
        
        # 4. 토큰 저장
        await db.add(download_token)
        await db.commit()
        
        # 5. 다운로드 이력 기록
        await self._log_download(album_id, user_id)
        
        return download_token

    async def _prepare_download_async(self, token: DownloadToken):
        """백그라운드에서 ZIP 파일 생성"""
        try:
            zip_data = await self.create_album_zip(token.album_id)
            
            # S3에 임시 업로드
            temp_url = await self._upload_to_s3(
                zip_data,
                f"downloads/{token.id}.zip"
            )
            
            token.status = "ready"
            token.download_url = temp_url
            token.file_size = len(zip_data)
            
            await db.commit()
            
            # 사용자에게 알림
            await self._notify_user(token.user_id, token.album_id)
            
        except Exception as e:
            logger.error(f"Download preparation failed: {e}")
            token.status = "failed"
            token.error_message = str(e)
            await db.commit()

    async def download_file(self, token_id: str) -> bytes:
        """다운로드 토큰으로 파일 다운로드"""
        
        token = await db.get(DownloadToken, token_id)
        
        # 1. 토큰 유효성 확인
        if token.status != "ready":
            raise ValueError(f"Download is {token.status}")
        
        if token.is_expired():
            raise ValueError("Download link expired")
        
        # 2. 파일 반환
        if token.file_data:
            # 메모리에 저장된 경우
            return token.file_data
        elif token.download_url:
            # S3에 저장된 경우
            return await self._download_from_s3(token.download_url)
        else:
            raise ValueError("Download file not found")

    def _can_download(self, album: Album, user_id: str) -> bool:
        """다운로드 권한 확인"""
        
        # 앨범 소유자는 항상 다운로드 가능
        if album.user_id == user_id:
            return True
        
        # 구독 플랜별 다운로드 제한 (선택)
        # user = await db.get(User, user_id)
        # if user.subscription == "studio":
        #     return True
        
        return False
```

### 5.3 다운로드 임시 저장소 정리

```python
class DownloadCleanupService:
    
    async def cleanup_expired_downloads(self):
        """24시간 이상 된 임시 다운로드 파일 삭제"""
        
        # 매일 자정에 실행 (Celery Beat 또는 APScheduler)
        
        expired_tokens = await db.execute(
            select(DownloadToken).where(
                DownloadToken.expires_at < datetime.utcnow()
            )
        )
        
        for token in expired_tokens:
            # S3에서 삭제
            if token.download_url:
                await self._delete_from_s3(token.download_url)
            
            # DB에서 삭제
            await db.delete(token)
            await db.commit()
            
            logger.info(f"Cleaned up download token: {token.id}")
```

```sql
    id VARCHAR(36) PRIMARY KEY,
    
    -- 대상 정보
    album_id VARCHAR(36) NOT NULL,
    user_id VARCHAR(36) NOT NULL,
    
    -- 파일 정보
    file_data LONGBLOB,                 -- 작은 파일용
    download_url VARCHAR(500),          -- S3 URL (큰 파일용)
    file_size BIGINT,
    
    -- 상태
    status VARCHAR(50),                 -- preparing, ready, failed
    error_message VARCHAR(500),
    
    -- 만료
    expires_at TIMESTAMP NOT NULL,
    
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    
    FOREIGN KEY (album_id) REFERENCES albums(id),
    FOREIGN KEY (user_id) REFERENCES users(id),
    
    INDEX idx_album_id (album_id),
    INDEX idx_user_id (user_id),
    INDEX idx_expires_at (expires_at)
);
```
